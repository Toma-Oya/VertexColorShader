# 前置き
ポートフォリオとして利用しているリポジトリです。
そのため、一般的な利用はあまり想定しておりません。

# 概要
Unityで頂点色を表示するシェーダーです。  
通常の頂点シェーダーとして利用可能な他、テクスチャとして白黒画像を設定することで、筆の柄のような模様をつけることが可能です。
